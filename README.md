# Active Directory Homelab
Built a Windows Server 2022 and Windows 11 Active Directory homelab using Oracle VirtualBox to gain hands-on experience with common IT support and system administration tasks. The environment includes Active Directory, DNS, Group Policy, shared drives, user account management, and endpoint administration.


## Environment
- Oracle VirtualBox
- Windows Server 2022
- Windows 11 Pro
- Active Directory
- DNS
- Group Policy
- Action1
